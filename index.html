Ordered Lists

There are different types of Ordered Lists, they're numbered by default.


Type 1 = Numerical (default).
Type A= The list will be numbered with uppercase letters.
type a= The list will be numbered with lowercase letters.
Type I=The list items will be numbered with uppercase roman numbers.
type="i" The list items will be numbered with lowercase roman numbers.
An example of what each ordered list looks like:
Type 1:

  1. Coffee
  2. Tea
  3. Milk

Type A:
  1. Coffee
  2. Tea
  3. Milk

Type a
  1. Coffee
  2. Tea
  3. Milk

Type I
  1. Coffee
  2. Tea
  3. Milk

Type i
  1. Coffee
  2. Tea
  3. Milk

Unordered Lists

There are different types of unordered lists and uses CSS life-style-type property to define the style of the marker.

Anchor tags


The tag defines a hyperlink. which is used to link pages together. The most important attribute of the tag is the href element which indicates the destination of the link.
Visit W3Schools.com! By default, links will appear as follows in all browsers:

  1. An unvisited link is underlined and blue
  2. A visited link is underlined and purple
  3. An active link is underlined and red
